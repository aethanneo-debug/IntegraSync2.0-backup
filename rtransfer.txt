TRANSFERRING YOUR PROJECT TO YOUR LOCAL MACHINE
=================================================

This is a comprehensive guide to transferring your project from this environment to your local machine. This guide covers how to set up the React/Vite full-stack application and its backend in your local environment.

Since you are exporting this project directly from AI Studio, you don't need to manually recreate any folder layouts or copy-paste code! The ZIP file or repository you download already contains everything.

Phase 1: Download the Code
--------------------------
1. Export the ZIP or clone the repository containing your project code.
2. Extract the downloaded ZIP file to any directory on your computer (for example, your Desktop).


Phase 2: Preparations
---------------------
Before you start, ensure you have these tools installed on your computer:
1. Node.js (v18 or higher) - Includes "npm"
2. Python 3 (v3.9 or higher) (Optional: if booting the companion backend_fastapi project)


Phase 3: Opening in Visual Studio Code
--------------------------------------
1. Open VS Code.
2. Go to File > "Open Folder..." and select the extracted project folder.
3. Open a new built-in terminal in VS Code (Terminal > New Terminal, or press `Ctrl + \``).


Phase 4: Install Dependencies
-----------------------------
In the VS Code terminal, run the following command to install all frontend and backend development dependencies:

    npm install


Phase 5: Running the Full-Stack App
-----------------------------------
Once npm finishes installing, start the local development server by running:

    npm run dev

This will boot up the full-stack development instance and bind to port 3000. Open your browser and navigate to:

    http://localhost:3000

Note: On the first time the server starts, the database will automatically be initialized, complete with default accounts and sample data as active in AI Studio.
